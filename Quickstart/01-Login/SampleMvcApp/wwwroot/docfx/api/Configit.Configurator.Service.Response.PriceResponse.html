    Class PriceResponse

    Response object returned by PriceAsync(PriceRequest).

    The response includes a pricing sheet with price lines and totals, the arguments used by the service, and (optionally) debug information.

    PackagePath

    The versioned package path used for generating this response.

    Even though the Configit.Configurator.Service.Request.IRequest.PackagePath is unversioned, this property will always contain the versioned path.

    Subsequent requests within the same configuration session should use the same versioned package path.
